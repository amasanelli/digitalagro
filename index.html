Bienvenido

Soy Ingeniero Agrónomo y Analista en Sistemas y me dedico a impulsar el crecimiento y eficiencia del sector agropecuario mediante soluciones tecnológicas personalizadas.

Con una sólida experiencia en agronomía y programación, desarrollo soluciones a medida que se adaptan a las necesidades específicas de empresas agropecuarias. Mi objetivo es integrar tecnología y campo para optimizar procesos, aumentar la productividad y facilitar la toma de decisiones basada en datos precisos.

Si buscas digitalizar tus operaciones, mejorar el control de tus cultivos o gestionar tu producción agrícola de manera más eficiente, estás en el lugar correcto.

¡Hagamos crecer tu negocio con la tecnología adecuada!

Proyectos destacados

Agricultura de precisión

Desde 2018, se han ambientado más de 100.000 hectáreas en la provincia de Buenos Aires. Se utilizan imágenes satelitales de Sentinel y Landsat, combinadas con mapas de rendimiento y evaluaciones de riesgos como inundaciones y heladas. Se desarrollan prescripciones de siembra y fertilización adaptadas a equipos Agrometal, Plantium, Precision Planting, entre otros.

Seguimiento de cultivos

Aplicación web para monitorear índices vegetales en tiempo real durante todo el ciclo de los cultivos. Ofrece análisis de imágenes y envía reportes semanales por correo electrónico con alertas personalizadas.

Costos de aplicación

Aplicación móvil para calcular el costo de pulverizaciones. Permite seleccionar insumos y dosis, e integra precios del ERP Albor, facilitando la toma de decisiones económicas.

Análisis de ensayos

Extensión para ArcGIS que automatiza el análisis de mapas de rendimiento. Facilita el cálculo de resultados en prescripciones variables mediante muestreo aleatorio y ANOVA.

Reportes en Power BI

Desarrollo de reportes interactivos en Power BI, integrando datos del ERP Albor y hojas de cálculo para monitorear siembra, fertilización y cosecha en tiempo real.

Avance de cosecha

Aplicación móvil para generar cartas de porte en el campo. Almacena datos para reportes de cosecha y permite registrar kilos almacenados en silos o bolsas.

Pedido de insumos

Plataforma móvil y web para gestionar pedidos de insumos. Se integra con el ERP Albor para controlar stock disponible y facilita la comunicación entre áreas.

Pedido de camiones

Aplicación para coordinar pedidos de transporte durante la cosecha. Permite realizar solicitudes al área de logística y recibir cupos asignados de manera eficiente.

Recetas agronómicas

Aplicación móvil para generar recetas agronómicas digitales en el campo. Facilita la carga de datos y permite descargar y compartir las recetas en formato PDF.

Cuaderno virtual

Aplicación móvil para registrar lluvias, recorridas y otros datos de interés agronómico. Se integra con sistemas de imágenes satelitales para mostrar índices vegetales actualizados.
